v0.1a1

2026 06/15
  • 建立 ar.html 與 ar/ar-scene.js，使用 MindAR + Three.js 製作秘密花園 WebAR 開花體驗。
  • 加入 image target 錨點、花瓣雨、花朵生長、柔和光暈、掃描提示與停止 AR 控制。
  • 加入 targets.mind 檢查、相機權限檢查、HTTPS 提醒與更清楚的啟動錯誤訊息。
  • 修正 MindAR/Three.js module 載入方式，支援 GitHub Pages、Netlify 與一般靜態伺服器。
  • 調整 AR 物件座標到圖卡附近，並新增辨識逾時提醒，方便現場確認是否真的觸發圖卡辨識。
  • 新增現場 debug log 與紅色方塊測試物件，用來判斷是否真正觸發目標圖片辨識。
  • 新增 ar-test.html 最小測試頁，並將主 AR 啟動流程改回 MindAR 預設相機與 UI 設定。
